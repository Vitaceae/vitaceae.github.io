docker 操作紀錄

# Set up docker environment 

$ sudo apt install qemu-system-misc qemu-user-static qemu-user binfmt-support
  • -it: 使用互動式終端機操作容器(interactive + tty)
  • –rm: 退出後刪除
  • -v $(pwd):/mnt: 共用資料(Volume Mount)；映射本機當前工作目錄到容器目錄 /mnt
  • DOCKER_REPO: 私有或公共伺服器網址
  • IMAGE_NAME
  • 1.0: 標籤(tag)
$ docker run --rm -it -v $(pwd):/mnt $DOCKER_REPO/$IMAGE_NAME:1.0 /bin/bash

without mapping

$ docker run --rm -it $DOCKER_REPO/$IMAGE_NAME:1.0 /bin/bash

退出容器

$ exit

# Remap user/group(optional)

Remap user/group between host and Docker container.
Files creted outside the container are owned by docker_user.

$ sudo adduser docker_user
$ cat /etc/subuid
docker_user:231072:65536
$ cat /etc/subgid
docker_user:231072:65536
$ sudo vi /etc/docker/daemon.json

{
    "userns-remap": "docker_user"
}

$ sudo systemctl restart docker.service

# Set up docker on local WSL 

$ sudo apt install apt-transport-https ca-certificates curl gnupg-agent software-properties-common
$ sudo apt install docker.io

Proxy 設定依佈署環境修改

/etc/docker/daemon.json

{
        "proxies": {
                "http-proxy": "http://localhost:port",
                "https-proxy": "http://localhost:port",
                "no-proxy": "localhost,my-domain"
        }
 
}

~/.docker/config.json

$ vi ~/.docker/config.json

{
        "proxies": {
                "default": {
                        "httpProxy": "http://ip:port",
                        "httpsProxy": "http://ip:port",
                        "no-proxy": "localhost,my-domain"
                }
        }
}

$ LINUX_USER=
$ sudo systemctl enable --now docker
$ sudo usermod -aG docker $LINUX_USER

Test

$ docker run hello-world

# Login into docker registry
$ docker login https://my-docker-server

# Problems

permission denied while trying to connect to the Docker daemon socket at unix:///var/run/docker.sock

$ sudo usermod -aG docker $USER

docker: Error response from daemon: Get “https://…”: tls: failed to verify certificate: x509: certificate signed by unknown authority.

需安裝憑證

# install certificate
# /etc/ssl/certs or /usr/local/share/ca-certificates/
$ sudo cp 'foo.cer' /usr/local/share/ca-certificates/'foo.crt'
$ sudo update-ca-certificates
$ sudo systemctl restart docker
