# Ubuntu Desktop Setup

Windows 11 更新卡死又要求 TPM 才能重新安裝, 怒而放生重拾 Ubuntu 20.04LTS Desktop
紀錄方便日後重新部署

Hotkeys Description
PrtScn 開啟截圖工具
Alt + PrtScn 視窗截圖
Window + Space 切換輸入法
Alt + Num terminal 切換 page

## Gparted

建立分割區並格式化, 懶得用 fdisk 等命令列工具
但不支援管理 LLM 或 RAID

## Chewing

安裝新酷音輸入法

$ sudo apt install ibus-chewing
$ reboot

進入設定: Keyboard > Input Sources
按 + 新增 Chinese (Chewing)

更改中英輸入切換快捷鍵

## Notepad++

### 調整 menu bar 功能表字體大小

預設字體太小十分傷眼
Notepad++ 非 Linux 原生應用程式而是透過 Wine 啟動, 所以要調整 Wine pixel 設定

$ notepad-plus-plus.wine winecfg

### 調整文字編輯區預設字型

安裝 Consolas 字型: 參考 sigoden/install-consolas-ubuntu
另存成 script 執行後 reboot

wget -O /tmp/YaHei.Consolas.1.12.zip https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/uigroupcode/YaHei.Consolas.1.12.zip
unzip /tmp/YaHei.Consolas.1.12.zip
sudo mkdir -p /usr/share/fonts/consolas
sudo mv YaHei.Consolas.1.12.ttf /usr/share/fonts/consolas/
sudo chmod 644 /usr/share/fonts/consolas/YaHei.Consolas.1.12.ttf
cd /usr/share/fonts/consolas
sudo mkfontscale && sudo mkfontdir && sudo fc-cache -fv

另一個字體來源

$ wget https://img.yangsihan.com/YaHeiConsolas.tar.gz

## Onedrive

預設來源的 oedrive 版本太舊

$ apt install onedrive

refer to Installation of ‘onedrive’ package on Debian and Ubuntu
採用新來源安裝

# Add the OpenSuSE Build Service repository release key
$ wget -qO - https://download.opensuse.org/repositories/home:/npreining:/debian-ubuntu-onedrive/xUbuntu_24.10/Release.key | gpg --dearmor | sudo tee /usr/share/keyrings/obs-onedrive.gpg > /dev/null
# Add the OpenSuSE Build Service repository
$ echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/obs-onedrive.gpg] https://download.opensuse.org/repositories/home:/npreining:/debian-ubuntu-onedrive/xUbuntu_24.10/ ./" | sudo tee /etc/apt/sources.list.d/onedrive.list
# Update your apt package cache
$ sudo apt-get update
# Install onedrive
$ sudo apt install --no-install-recommends --no-install-suggests onedrive

授權存取

$ onedrive

# 以瀏覽器開啟系統顯示的 URL, 登錄 Microsoft 帳戶並授權 OneDrive client
# 將產生的程式碼貼到 terminal

# once there's SLL error
$ onedrive --force-http-11
$ sudo update-ca-certificates

設定目錄

$ vi ~/.config/onedrive/config

sync_dir = "backup/onedrive"
log_dir = "~/.config/onedrive/logs"
# sec
#monitor_interval = "3600"
#monitor_fullscan_frequency = "12"
# deprecated?
#no_remote_delete = "true"
#upload_only = "false"
download_only = "true"
#rate_limit = "0" #KB/s
#skip_file_prefixes = "~,.~,.~lock."
#skip_file_suffixes = ".tmp,.temp,.swp"

設定需要同步的目錄(optional; case-sensitive)

$ vi ~/.config/onedrive/sync_list

onedrive 命令

# 測試
$ onedrive --dry-run --synchronize

# 同步
# opt:
#   --local-first: 先上傳後下載
#   --single-directory 'My_Dir': 同步指定目錄 My_Dir
#   --synchronize --upload-only: 僅上傳
$ onedrive --synchronize [opt]

自動啟動服務
$ systemctl --user enable onedrive
$ systemctl --user start onedrive
$ loginctl enable-linger $USER
