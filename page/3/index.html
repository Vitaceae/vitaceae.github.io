# Hugo 做法

  1. 在主題模板的 layouts/partials/head.html 中插入
<!--
{% ... %} template tags
{{ ... }} template variables
以下敘述判斷 .toml 檔案中 params.UmamiAnalytics 的 enabled 值決定是否引入外部 html
-->

{{ if (.Site.Params.UmamiAnalytics.enabled ) }}
    {{ partial "umami_analytics.html" . }}
{{ end }}

# Google Analytics

  1. 建立 Google Analytics 帳戶
    • 帳戶名稱: (任意)
    • 屬性名稱: Github account(任意)
    • …
    • 平台類型選擇「網站」
    • 網站網址: https://vitaceae.github.io
    • 串流名稱: Mike’s Blog(任意)
  2. 取得 Google Analytics 追蹤碼
  3. 覆蓋 Hugo 主題設定
    • 嵌入 gtag.js 到 anubis2 主題路徑 layouts/partials/google_analytics.html
    • hugo.toml 
      [params.GoogleAnalytics]
enabled = true
gtag = "G-P1Z223FHVN"

# Emoji

  • use emojify function and Inline Shortcodes to type emoji directly in content files
    • configuration requirements: set enableEmoji to true
    • will enable Unicode Standard emoji characters and sequences
    • the rendering depends on the browser/platform
    • e.g. 
      I :heart: you
      I ❤️ you
  • Emoji cheat sheet
  • use a 3rd party emoji font or a font stack 
    .emoji {
  font-family: Android Emoji, EmojiSymbols;
}
