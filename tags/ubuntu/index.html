# Ubuntu Desktop Setup

Windows 11 更新卡死又要求 TPM 才能重新安裝, 怒而放生重拾 Ubuntu 20.04LTS Desktop
紀錄方便日後重新部署

Hotkeys Description
PrtScn 開啟截圖工具
Alt + PrtScn 視窗截圖
Window + Space 切換輸入法
Alt + Num terminal 切換 page

## Gparted

建立分割區並格式化, 懶得用 fdisk 等命令列工具
但不支援管理 LLM 或 RAID

## Chewing

安裝新酷音輸入法

$ sudo apt install ibus-chewing
$ reboot

進入設定: Keyboard > Input Sources
按 + 新增 Chinese (Chewing)

更改中英輸入切換快捷鍵
