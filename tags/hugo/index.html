# Emoji

  • use emojify function and Inline Shortcodes to type emoji directly in content files
    • configuration requirements: set enableEmoji to true
    • will enable Unicode Standard emoji characters and sequences
    • the rendering depends on the browser/platform
    • e.g. 
      I :heart: you
      I ❤️ you

  • use a 3rd party emoji font or a font stack

# global-wise style 

<cust>
affected by <code>cust.scss</code> and <code>cust.css</code>
</cust>

<div class="cust-style">
example of <code>cust-style</code>
</div>
affected by cust.scss and cust.css
example of cust-style

add the following line to .toml

customCSS = ["custom-css/cust.scss", "custom-css/cust.css"]

add assets/custom-css/cust.css

.cust-style {
    background-color: pink;
    color: black;
    padding: 1em;
    border-radius: 0.3em;
    border: 1px solid black;
}

add assets/custom-css/cust.scss

@import 'foundation/vars';

cust {
    display: block;
    font: 100% $font-stack;
    color: $primary-color;
    background-color: $background-color;
    margin-top: 1em;
}

add assets/custom-css/foundation/_vars.scss

$font-stack: Helvetica, sans-serif;
$primary-color: rgb(255, 255, 168);
$background-color: rgb(85, 85, 85);

## Linux 

$ sudo snap install hugo

# enable/revoke access to removable media
$ sudo snap connect hugo:removable-media
$ sudo snap disconnect hugo:removable-media

# enable/revoke access to SSH keys
$ sudo snap connect hugo:ssh-keys
$ sudo snap disconnect hugo:ssh-keys

# Quick Start

  • Hugo: Basic usage
  • Windows user: run commands from PowerShell(not Windows PowerShell), WSL or Git Bash

## Create a demo site on WSL 

$ SITE_NAME=hugo_site
$ hugo new site $SITE_NAME
$ cd $SITE_NAME
$ git init

# add a new theme
$ git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
$ echo "theme = 'ananke'" >> hugo.toml

# write a new post; set 'draft = false' in the .md file when done
$ hugo new posts/new_post.md

# need to find the WSL's LAN IP; add an alias function in bash.rc
function hugo_server() {
    WSL_IP=$(ip addr show eth0 | grep "inet\b" | awk '{print $2}' | cut -d/ -f1)
    hugo server --bind $WSL_IP --baseURL=http://$WSL_IP -D -F --gc -w
}

# run local web server(changes take effect on the fly)
$ hugo_server
